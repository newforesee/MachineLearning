package core

Package defined for the core component of Scala for Machine learning. The package defines the following components:

- Basic matrix operations '''Matrix'''
- Types aliases and types conversion '''Types'''
- Monad for data transformation '''Monad'''
- Data transformation with explicit configuration '''ETransform'''
- Data transformation with model implicitly extracted from data  '''ITransform'''
- Design pattern for classifiers '''Design.model'''and '''Design.config'''

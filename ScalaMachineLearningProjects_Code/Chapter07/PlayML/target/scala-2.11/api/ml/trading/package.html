package trading

This package object that contains the constructs and tools for trading securities:

- Progammable representation of option on securities OptionModel

- Source for daily trading data on specific stocks for technical analysis YahooFinancials and GoogleFinancials

- Source for corporate data used in stock fundamental analysis Fundamentals

- Implementation of trading signals or events Signal

- Trading strategies as combination of signals TradingStrategy - Automated generation of trading strategies StrategyFactory

  1. final class OptionModel extends AnyRef

    Class that defines the model for a traded option on a security.

    Class that defines the model for a traded option on a security. An option is fully defined with the symbol of the underlying security, its strike price, the source of data (CSV format), the minimum expiration time left in the option and the number of steps used in the approximation of the value of the option (Discretization).

Related Doc: package qlearning

case class QLModel(bestPolicy: QLPolicy, coverage: Double) extends Model[QLModel] with Product with Serializable

Define a model for the Q-learning algorithm as the tuple <optimum policy, training epoch coverage>.

bestPolicy

Best policy computed or estimated during training.

coverage

Ratio of training trial or epochs that reach a predefined goal state.
